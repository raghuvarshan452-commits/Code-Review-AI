===========================================
ANIMATIONS & TRANSITIONS
===========================================

Page transitions (React Router):
Wrap each page in:
<div className="fade-slide-up" 
     style={{ animationDuration: '0.4s' }}>

Stagger issue cards:
Apply stagger-1 through stagger-5 classes
For cards beyond 5: use inline style
animationDelay: `${index * 0.06}s`
opacity: 0 (let fade-slide-up handle it)

Score count-up:
useEffect on review complete
requestAnimationFrame loop
increment from 0 to final score over 1200ms
easeOutQuart easing:
t => 1 - Math.pow(1 - t, 4)

Progress bar fill:
On review start: animate width 0 → 85% 
over 8000ms cubic-bezier(0.1, 0.4, 0.8, 1.0)
On complete: transition width to 100% over 300ms
Then fade out progress bar opacity 0 over 500ms

Sidebar stat bars:
On mount: animate each bar width from 0 
to calculated % over 600ms ease
staggered by 100ms each
